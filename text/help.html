Справка по программе Метод Анализа Иерархий.

Рабочая область программа разделена на несколько вкладок, на которые вынесены отдельные этапы метода анализа иерархий.

Вкладка Параметры

На данной вкладке вы можете задать количество Критериев и Альтернатив, изменяя значения в соответствующих полях. Дважды кликнув по названию Критерия или Альтернативы можно изменить это значение.

Вкладки Парных сравнений

На данных вкладках представлена матрица парных сравнений Критериев между собой или Альтернатив по какому то критерию. Дважды кликнув по ячейке в таблицы можно изменить значение парного сравнения, выбрав его выпадающего списка. В нижней части представлена информация о значении Отношения согласованности.

Вкладка результат

На данной вкладке находится матрица приоритетов Альтернатив по Критериям. Зеленным цветом выделяется ячейка лучшей Альтернативы по заданному критерию. В нижней части представлена Лучшая альтернатива на основе всех критериев.
